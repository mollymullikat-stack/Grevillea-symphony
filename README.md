# 🎶 Grevillea Symphony

> **"When every bloom plays its season"**

แอปบันทึกโปรไฟล์ต้นเกรวิลเลียในสวน **Buay Bloom Garden** — สีสันสดใส ใช้งานง่าย เก็บข้อมูลในเครื่อง (localStorage) ไม่ต้องต่อเน็ต

A colorful single-file web app for cataloguing the Grevillea collection of Buay Bloom Garden.

## ✨ ฟีเจอร์ / Features

- 🌿 โปรไฟล์แต่ละต้นครบ 4 หมวด: ข้อมูลพื้นฐาน+รูป · ช่วงออกดอก · การดูแล · ขนาด & การเติบโต
- 🎼 ระบบ **Tree ID** (G01–G23) + บทบาทในวงดนตรี (Instrument)
- จัดกลุ่มเป็น 3 หมวด: 🤍 **Silver Symphony** · 🧡 **Warm Bloom** · ❤️‍🔥 **Bold Drama**
- 🔍 ค้นหา + กรองตามหมวด / สถานะดอก
- 🖊️ เพิ่ม / แก้ไข / ลบ ต้นได้เองในเว็บ พร้อมอัปโหลดรูปต้นจริง
- 📅 แถบฤดูกาลออกดอก 12 เดือนแบบ visual
- มาพร้อมคอลเลกชันตั้งต้น 23 ต้นของ Buay Bloom Garden

## 🚀 วิธีใช้ / Usage

เปิดไฟล์ [`grevillea-symphony.html`](grevillea-symphony.html) ด้วยเบราว์เซอร์ได้เลย (ดับเบิลคลิก หรือเปิดผ่าน Chrome)
